# Exp001 - Minimal CLIP-based IQA Pipeline

## Goal

Build a minimal CLIP-based IQA training pipeline from scratch using the CSIQ dataset.

---

# Pipeline

```text
CSIQ Dataset
↓
Dataset
↓
DataLoader
↓
CLIP Image Encoder
↓
MLP Regressor
↓
Quality Score
↓
MSE Loss
```

---

# What I Implemented

## 1. CSIQ Dataset

Implemented a custom `CSIQDataset` class:

- Read image paths and MOS labels from `csiq_label.txt`
- Loaded images using PIL
- Converted images to RGB
- Applied transforms:
  - Resize(224,224)
  - ToTensor()

Returned:

```python
image, score
```

---

## 2. DataLoader

Built DataLoader with:

```python
batch_size = 4
shuffle = True
num_workers = 0
```

Verified tensor shapes:

```python
images.shape = [4, 3, 224, 224]
scores.shape = [4]
```

---

## 3. CLIP-based IQA Model

Implemented a minimal CLIP-IQA model:

```text
Image
↓
CLIP ViT-B/32 Encoder
↓
512-d Image Feature
↓
MLP Regressor
↓
Predicted Quality Score
```

Model structure:

```python
Linear(512,128)
ReLU
Linear(128,1)
```

Frozen CLIP parameters and trained only the regression head.

---

## 4. Training Pipeline

Implemented:

- MSE Loss
- Adam Optimizer
- Forward propagation
- Backward propagation
- Gradient update

Training loop:

```python
optimizer.zero_grad()
loss.backward()
optimizer.step()
```

# Key Understandings

## Dataset vs DataLoader

### Dataset

Defines how to read a single sample:

```python
image, score
```

### DataLoader

Automatically builds mini-batches:

```python
[B, 3, 224, 224]
```

---

## Transform

Learned that transforms convert:

```text
PIL Image
↓
Tensor
```

and resize images into CLIP-compatible input format.

---

## CLIP Feature Extraction

Learned that:

```python
clip_model.encode_image()
```

extracts semantic image features:

```python
[B, 512]
```

---

## Training Mechanism

Understood:

```python
loss.backward()
```

computes gradients automatically.

Understood:

```python
optimizer.step()
```

updates model parameters using gradients.

Understood why:

```python
optimizer.zero_grad()
```

is necessary to clear accumulated gradients.
